# Guía 4: Tipos de Estructuras de Datos

Strings, listas, tuplas y diccionarios. La guía más larga y densa de la materia.

---

## Temas

- Strings: slicing, métodos, `map`, `filter`
- Rangos, tuplas, listas
- Listas anidadas y matrices
- Diccionarios: creación, acceso, iteración, operaciones
- `map` y `filter`
- Ordenamiento con `sorted`
