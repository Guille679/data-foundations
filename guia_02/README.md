# Guía 2: Tipos de Datos, Expresiones y Funciones

Primeros scripts en Python. Foco en tipos de datos, operadores, variables y definición de funciones con parámetros y `return`.

---

## Temas

- Tipos de datos: `int`, `float`, `str`, `bool`
- Variables y asignación
- Operadores numéricos y de texto
- Casteo: `int()`, `float()`, `str()`
- Definición de funciones: `def`, parámetros, `return`
- `input` y `print`
- Slicing de strings
- Funciones predefinidas: `abs`, `len`, `max`, `min`, `round`
- Buenas prácticas: snake_case, nombres descriptivos
