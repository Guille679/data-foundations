## Ejercicio 7 — Instrucciones para usar un cajero automático

**EJERCICIO DADO**
Querés explicarle a una persona que nunca usó un cajero automático cómo extraer dinero. Describí paso a paso todas las instrucciones necesarias, considerando que la persona no sabe nada sobre cajeros. ¿Qué información necesita tener antes de empezar? ¿Qué decisiones debe tomar durante el proceso?

**Objetivo:** Guiar a un usuario principiante en el proceso de extraer dinero físico de forma segura y exitosa.

**Elementos necesarios:**

**REQUISITOS PREVIOS (Antes de empezar)**
Para que la operación sea posible, el usuario debe contar con:

- Tarjeta de Débito: El plástico físico con chip o banda magnética.
- Clave PIN: El código numérico de 4 dígitos (secreto).
- Saldo Disponible: Saber que tiene dinero en la cuenta.
- Ubicación del Cajero: Identificar la red (Banelco, Link, etc.).

**DECISIONES DURANTE EL PROCESO**
- Tipo de cuenta: Elegir entre Caja de Ahorro o Cuenta Corriente.
- Monto a extraer: Decidir una de las opciones fijas o ingresar un monto personalizado.
- Comprobante: Elegir si quiere imprimir el ticket de papel o verlo en pantalla.

**Pasos:**
1. Insertar la tarjeta: Introducir el plástico en la ranura (generalmente con el chip hacia arriba y adelante).
2. Ingresar PIN: Digitar el código de 4 dígitos.
   - Si es correcto: continuar.
   - Si es incorrecto: el sistema permite reintentar.
   - Si falla 3 veces consecutivas: la tarjeta queda bloqueada y el proceso termina.
3. Seleccionar Operación: Elegir la opción "Extracciones/Adelantos" en el menú principal.
4. Configurar monto: Seleccionar el tipo de cuenta y la cantidad de dinero deseada.
5. Retirar valores en este orden:
   a. Retirar el dinero de la ranura inferior.
   b. Retirar la tarjeta cuando el cajero la expulse.
   c. Retirar el comprobante si se solicitó.
