## Ejercicio 5 — Registro de Estudiantes

**EJERCICIO DADO**
Se te encargó definir qué datos son necesarios para el registro de estudiantes en un curso de inglés. ¿Qué datos crees que deberían ser obligatorios y cuáles opcionales? ¿Y si el curso es de cocina?

**Objetivo:** Definir y categorizar la información necesaria para inscribir alumnos en diferentes tipos de formación (Inglés y Cocina).

**Elementos necesarios:**

**PARA CURSO DE INGLES**
Para un idioma, lo vital es el nivel previo y la comunicación.

Datos Obligatorios:
- Nombre y Apellido: Identificación básica.
- DNI / Pasaporte: Para certificación oficial.
- Email / Teléfono: Canal de envío de material y avisos.
- Nivel de inglés actual: (Principiante, Intermedio, Avanzado) para asignar el aula correcta.

Datos Opcionales:
- Razon del estudio: (Trabajo, estudio, hobby) 
- Conocimiento de otros idiomas: Ayuda a entender el perfil de aprendizaje.
- Disponibilidad horaria: Si hay varios turnos disponibles.

**PARA CURSO DE COCINA**
Aquí los datos cambian porque hay un factor de seguridad física (alergias) y logística (insumos).

Datos Obligatorios:
- Nombre y Apellido: Identificación básica.
- Teléfono de emergencia: Crucial en un ambiente con fuego y cuchillos.
- Restricciones alimentarias / Alergias: (Celíacos, alergia a frutos secos, etc.) para evitar accidentes graves.
- Experiencia previa en cocina: Para nivelar el uso de herramientas.

Datos Opcionales:
- Preferencia de dieta: (Vegano, vegetariano) para adaptar recetas.
- Talle de delantal/chaquetilla: Si el curso provee uniforme.
- Redes sociales: Para compartir fotos de los platos terminados.

**Pasos:**
1. Presentar el formulario al estudiante.
2. Validar que los campos obligatorios no estén vacíos.
3. Verificar el formato del email (que tenga un @).
4. Almacenar los datos en la base de datos del curso.
5. Emitir un comprobante de inscripción con los datos cargados.
