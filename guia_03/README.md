# Guía 3: Estructuras de Control

Condicionales y ciclos. A partir de acá, todo se resuelve dentro de funciones.

---

## Temas

- Expresiones booleanas y operadores de comparación
- `if`, `elif`, `else`
- Ciclo `for` y `range`
- Ciclo `while`
- `break`, `continue`, `return` dentro de ciclos
- Buenas prácticas con `while`: estado previo, condición, paso
