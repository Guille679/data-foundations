# 📊 Unidad 6: Matplotlib

Ejercicios y scripts sobre visualización de datos con Matplotlib.

**Basado en:** Cátedra de Pensamiento Computacional — CBC, FIUBA

---

## 📁 Contenido

| Archivo | Tema |
|---|---|
| `01_graficos_basicos.py` | Line plot, bar chart, scatter plot |
| `02_personalizacion.py` | Títulos, etiquetas, leyendas, colores |
| `03_graficos_multiples.py` | Subplots y múltiples gráficos |
| `04_integracion.py` | Matplotlib + Pandas + NumPy |

---

## 🎯 Temas cubiertos

- Creación de figuras y ejes
- Tipos de gráficos: línea, barras, dispersión
- Personalización: títulos, referencias, colores
- Subplots — múltiples gráficos en una figura
- Integración con Pandas y NumPy
