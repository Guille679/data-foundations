# 🐼 Unidad 6: Pandas

Ejercicios y scripts sobre manipulación y análisis de datos con Pandas.

**Basado en:** Cátedra de Pensamiento Computacional — CBC, FIUBA

---

## 📁 Contenido

| Archivo | Tema |
|---|---|
| `01_series.py` | Series: creación, indexado, operaciones |
| `02_dataframes.py` | DataFrames: creación, acceso, modificación |
| `03_limpieza.py` | Limpieza de datos, valores nulos |
| `04_agregacion.py` | Agrupación y agregación de datos |

---

## 🎯 Temas cubiertos

- Series y DataFrames
- Acceso a filas y columnas
- Filtrado y selección de datos
- Limpieza: valores nulos, duplicados
- Agrupación con `groupby`
- Lectura de archivos CSV
