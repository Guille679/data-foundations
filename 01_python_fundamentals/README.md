# 🐍 Unidad 1–5: Fundamentos de Python

Ejercicios y scripts cubriendo los fundamentos del lenguaje Python, desde algoritmos básicos hasta manejo de archivos.

**Basado en:** Cátedra de Pensamiento Computacional — CBC, FIUBA

---

## 📁 Contenido

| Archivo | Tema |
|---|---|
| `01_tipos_y_variables.py` | Tipos de datos, variables, operadores, casteo |
| `02_funciones.py` | Definición de funciones, parámetros, return |
| `03_control_de_flujo.py` | if / elif / else, operadores lógicos y de comparación |
| `04_ciclos.py` | for, while, break, continue, range |
| `05_estructuras_de_datos.py` | Strings, listas, tuplas, diccionarios |
| `06_archivos.py` | Lectura y escritura de archivos, manejo de errores |

---

## 🎯 Temas cubiertos

- Algoritmos y pensamiento computacional
- Tipos de datos: `int`, `float`, `str`, `bool`
- Operadores numéricos, de texto y lógicos
- Funciones: definición, parámetros, return, buenas prácticas
- Estructuras de control: condicionales y ciclos
- Estructuras de datos: listas, tuplas, diccionarios, strings
- Entrada/salida: `input`, `print`, manejo de archivos
- Manejo de errores básico

---

## 🛠️ Requisitos

```bash
python 3.10+
```

Sin dependencias externas — solo Python estándar.
