# 📊 Data Foundations

Repositorio de práctica y proyectos construido durante la materia **Pensamiento Computacional** del CBC — UBA.  
Cubre Python desde los fundamentos hasta análisis de datos real con NumPy, Pandas y Matplotlib.

**Stack:** Python · NumPy · Pandas · Matplotlib · SQL · Power Bi · Excel

---

## 📁 Estructura

| Carpeta | Contenido |
|---|---|
| `guia_01/` | Algoritmia y descomposición de problemas — sin código |
| `guia_02/` | Tipos de datos, expresiones y funciones |
| `guia_03/` | Estructuras de control: condicionales y ciclos |
| `guia_04/` | Estructuras de datos: strings, listas, tuplas, diccionarios |
| `guia_05/` | Entrada/salida: archivos y manejo de errores |
| `guia_06/` | Bibliotecas: NumPy, Pandas, Matplotlib |
| `projects/` | Proyectos completos de análisis de datos |

---

## 🎯 Objetivo

Construir una base sólida en el ecosistema de datos de Python que cubra:

- Programación Python orientada al análisis de datos
- Manipulación y limpieza de datasets con Pandas
- Operaciones numéricas eficientes con NumPy
- Visualización de datos con Matplotlib
- Proyectos end-to-end sobre datos reales

---

## 🛠️ Requisitos

```bash
python 3.10+
```

Guías 1–5: sin dependencias externas.
